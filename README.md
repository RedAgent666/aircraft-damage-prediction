# aircraft-damage-prediction

# Прогнозирование степени повреждения воздушных судов (ML)

Реализация модели машинного обучения для предсказания степени повреждений самолетов на основе исторических данных. Проект выполнен в рамках аналитического трека обучения.

### 📊 Ключевые результаты
* Обучена модель бинарной/мультиклассовой классификации, продемонстрировавшая точность (**Accuracy**) на тестовой выборке в **79%**.
* Проведен полный цикл исследования данных (EDA), включая очистку данных, заполнение пропусков и генерацию новых признаков (Feature Engineering).
* Визуализированы распределения ошибок модели для выявления слабых зон алгоритма.

### 🛠 Стек технологий
* **Python** (библиотеки: Pandas, NumPy, Scikit-Learn)
* **Визуализация:** Seaborn, Matplotlib
* **Среда разработки:** Jupyter Notebook
